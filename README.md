> How much of your project is actually yours?

A web experiment built at **2025 GDG Hackathon** that analyzes npm dependencies and visualizes them as **3D parasitic organisms**.

Enter a GitHub repository URL, and it will show you the ratio between the code you wrote and the external packages your project depends on вЂ” rendered as a living ecosystem of creatures surrounding a tiny host nucleus.

---

## рџ§Є What it does

* Parses `package.json` from a public GitHub repository
* Collects 1-level transitive dependencies from the npm registry
* Measures your code size via GitHub Languages API
* Calculates `hostRatio` (your code vs. external code) and `dependencyAmplification`
* Classifies infestation severity:

> **Symbiotic** в†’ **Colonized** в†’ **Consumed** в†’ **Hollow Shell**

---

## рџ¦  Parasite Codex

Each dependency becomes a unique **biological specimen**, generated by Google's Gemini based on the package's characteristics.

* **Body**: sphere, ellipsoid, torus, capsule вЂ” combined with segments
* **Appendages**: rings, cones, tendrils attached in radial or random patterns
* **Motion**: breathing, pulsing, drifting, flickering вЂ” each creature moves differently
* **Size**: proportional to transitive dependency count
* **Color**: varies by dependency group

If Gemini generation fails, fallback variants are procedurally generated from the package name.

---

## рџЊЊ The Scene

After analysis, a 3D scene renders your project as a living organism.

* A small glowing nucleus at the center represents your code
* Parasites of varying shapes and sizes surround it
* Click a parasite to view its specimen card вЂ” species name, bio report, dependency info
* Press **"Show Host Only"** to strip away all parasites and reveal how small your code really is
