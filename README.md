# 🦠 Dependency Parasite

> Is your code… really yours?

`Dependency Parasite` began as a simple question.
How much external code is layered on top of what we write —
and how deeply does it seep into the core?

Give it a GitHub repository,
and it will dissect it quietly,
revealing the result as **softly writhing 3D parasitic organisms**.

At the center, a small and fragile host — your code.
Around it, countless entities drifting closer.

They are not yours.
But they are already a part of it.

---

## 🧪 What this experiment observes

Feed it a repository, and it will examine:

* Direct dependencies declared in `package.json`
* The 1-level transitive dependencies hidden behind them
* The amount of code you actually wrote (via GitHub Languages API)
* Your code's share of the whole (`hostRatio`)
* How much dependencies have multiplied (`dependencyAmplification`)

From this, the project is interpreted as a living ecosystem.

> Symbiotic / Colonized / Consumed / Hollow Shell

Which stage does yours belong to?
You'll have to see for yourself.

---

## 🦠 Parasite Codex

Each dependency is no longer just a number,
but transformed into a **biological specimen**.

Every parasite carries its own distinct form and behavior.

* **Body**: spherical, elongated, or something less familiar
* **Appendages**: rings, tendrils, fangs — structures that emerge and grow
* **Motion**: subtle breathing, pulsing, drifting through space
* **Size**: expands with the weight of its transitive dependencies
* **Color**: shifts depending on its dependency type

These organisms are generated by
Google's Gemini, interpreting the nature of each package.

If generation fails, no need to worry.
Primitive variants will still emerge — shaped from the trace of their names.

---

## 🌌 The Scene

When the analysis completes, a single scene unfolds.

At the center: a small nucleus — your code.
Around it… no explanation is necessary.

Click on a parasite to open its specimen card.
Species name, ecological report, transitive dependency count — all in one card.

And when you press **"Show Host Only"**,
every parasite vanishes, leaving only the nucleus at the center.
That is everything you wrote yourself.

* pnpm, ESLint
